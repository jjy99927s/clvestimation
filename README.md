# Customer Lifetime Value Prediction

This project predicts customer lifetime value using purchase frequency and
average order value with a Linear Regression model.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model
Linear Regression

## Dataset
Customer transaction dataset

## Output
Actual vs Predicted CLV visualization and model performance metrics.
